# MajTasks

## Apps

### Tasks UI

To serve the app:
```bash
npm run tasks-ui:start
```

To do the unit test of the app:
```bash
npm run tasks-ui:test
```

To run end to end test on the app:
```bash
npm run tasks-ui:e2e
```

### Tasks API

To serve the app:
```bash
npm run tasks-api:start
```

To do the unit test of the app:
```bash
npm run tasks-api:test
```

To run end to end test on the app:
```bash
npm run tasks-api:e2e
```
